# Sentifyd Avatar Google Tag Manager Template

This repository contains the Google Tag Manager template for the Sentifyd Avatar. This tag allows you to easily integrate the Sentifyd 3D AI Avatar into your website.

## Description

The Sentifyd Avatar tag adds a 3D AI Avatar to your website, enabling interactive chat and advertising experiences.

## Configuration

To use this template, you need an **Avatar API Key**.

1.  Sign up at [https://sentifyd.io](https://sentifyd.io).
2.  Create your avatar to generate an API key.
3.  Enter the API key in the tag configuration in Google Tag Manager.

## Documentation

For more detailed instructions and documentation, please visit [https://docs.sentifyd.io](https://docs.sentifyd.io).
